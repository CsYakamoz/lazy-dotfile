## How to use

1. `mkdir ~/.zsh`
2. `ln -s ./zsh/zshrc ~/.zsh/zshrc`
3. source `~/.zsh/zshrc` in `~/.zshrc`

   ```zsh
   source ~/.zsh/zshrc

   # Add your custom configurations here
   ```
